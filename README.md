# Rwaq-App
رواق (Rwaq) - المسودة الذكية بنظام الوصول الغامض ​تطبيق مسودة (Drafting App) مفتوح المصدر مصمم للمبدعين والمعماريين. يجمع بين الفن الرقمي والذكاء الاصطناعي مع تجربة مستخدم فريدة تعتمد على "الأمن بالتمويه". ​بوابة الوصول: واجهة دخول مخادعة "تهرب" من المستخدم، لا يمكن تجاوزها إلا برسم إيماءة
